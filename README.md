# Wabot Web
Interface para exibir os dados de envio e recebimento de mensagens

## Requisitos
- PHP 5.6 +
- MySQL
- Composer

## Instalação
- `composer install`
- Renomeie `.env.example` para `.env`
- Preencha os dados de conexão com o banco de dados
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan serve`
